# Simple Calculator GUI

A simple calculator application built with Python and Tkinter. The program allows users to enter two numbers, choose an arithmetic operation, and display the result through a graphical user interface (GUI).

## Features

- User-friendly graphical interface
- Input two numbers
- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Display the calculation result
- Error handling for invalid input
- Prevents division by zero

## Requirements

- Python 3.x
- Tkinter (included with standard Python installation)
